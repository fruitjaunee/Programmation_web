curl: “command line tool and library for transferring data with URLs”

HTTP debugging with curl

curl is a powerfull http (but not only) console client, it helps to explore and debug http programing interfaces.

Please RTFM:

  • curl.haxx.se/docs/manual.html !
  • curl.haxx.se/docs/httpscripting.html

Here are some common commands:

  • Request an URL and show response headers: 
    curl --head http://127.0.0.1:8000/
curl -i -X OPTIONS http://127.0.0.1:8000/
  • Get some data: 
    curl -is "https://fr.wikipedia.org/w/index.php?search=Alan+Turing" | grep location
  • Post some data (json payload): 
    curl -d '{"key1":"value1", "key2":"value2"}' -H "Content-Type: application/json" -X POST http://localhost:8800/data
curl -d "@data.json" -X POST http://localhost:8800/data
